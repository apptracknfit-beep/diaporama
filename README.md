# Diaporama

Diaporama responsive de **126 photos** classées par date (la plus ancienne d'abord).

- Ouvrir [`index.html`](index.html) ou la version déployée
- Navigation :
  - Flèches clavier `←` / `→`, `Espace`
  - Clic gauche / clic droit sur la photo
  - Swipe sur mobile
- Lecture auto disponible (icône ▶)
- Plein écran (`F` ou icône)
- Le cadre s'adapte à la taille naturelle de chaque photo : aucune déformation, aucune coupe
- 100 % responsive : téléphone, tablette, desktop

## Lancer en local

```bash
python -m http.server 8000
# puis http://localhost:8000
```
